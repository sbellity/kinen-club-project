# Methodology Reference

This session follows the kinen round-based iterative refinement methodology.

See the full methodology documentation: `kinen/docs/methodology.md`

## Key Principles

- **Round-based exploration**: Iterative questions and refinement (typically 5-6 rounds)
- **Document-first**: Substantive work in markdown documents, chat for coordination
- **Inline annotation**: Responses added using `>>` markers in round files
- **Living documents**: Primary document evolves through rounds
- **Concrete over abstract**: Specific examples and scenarios preferred

## Session Structure

- `init.md` - Goals and initialization
- `round-XX.md` - Iterative exploration rounds
- `requirements.md` - Living document (or `integration-plan.md` for this session)
- `session-summary.md` - Final summary (created at end)
